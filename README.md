# Portfolio

A **lightweight, mobile-first portfolio dashboard** engineered to resemble real-world **observability systems and CI/CD pipelines**.

This is not a typical portfolio.  
It is designed to **simulate how infrastructure engineers think, operate, and deliver at scale**.

---

## 🛠 Tech Stack

| Layer        | Technology |
|--------------|-----------|
| UI Framework | HTML + Tailwind CSS (CDN) |
| Styling      | Tailwind + Custom CSS |
| Icons        | Lucide Icons |
| Fonts        | Inter, JetBrains Mono |
| Architecture | Static SPA |
