Portfolio Website - Livio Tri Gieraldo
Repositori ini berisi kode sumber untuk website portofolio profesional saya, dibangun untuk menampilkan rekam jejak sebagai Cloud & DevOps Engineer. Proyek ini dikembangkan dengan pendekatan Plan-First dan diatur menggunakan framework RTCC-O.

📌 Project Overview
Website portofolio ini berfokus pada kecepatan, kebersihan kode, dan aksesibilitas. Alih-alih bergantung pada framework berat, portofolio ini dibangun menggunakan prinsip dasar web engineering untuk membuktikan pemahaman inti terhadap infrastruktur front-end.

🛠️ Technical Stack
Sesuai dengan Technical Requirements:
Semantic HTML5: Menyediakan struktur data yang solid.
Vanilla CSS3: Menangani semua tata letak dan desain responsif (Mobile-First Approach) tanpa library eksternal.

📂 Repository Structure
Struktur repositori ini mematuhi standar yang telah ditentukan:
.
├── README.md                 # Dokumentasi proyek (file ini)
├── index.html                # Struktur utama website (HTML5)
├── style.css                 # Aturan tata letak dan visual (CSS3)
├── assets/                   # Aset gambar dan tangkapan layar
│   ├── desktop-view.png      # Tampilan resolusi desktop (1024px+)
│   └── mobile-view.png       # Tampilan resolusi ponsel (375px)
└── plan/                     # Dokumentasi proses pengembangan
    ├── 01-brainstorm.md      # Rencana arsitektur dengan AI
    ├── 02-details.md         # Kerangka kerja RTCC-O
    ├── 03-execution.md       # Logika eksekusi langkah demi langkah
    └── 04-results.md         # Refleksi dan hasil validasi teknis


🚀 Deployment & Cara Menjalankan
Proyek ini bersifat statis (tanpa build tool atau dependensi Node.js).
    1. Lakukan clone pada repositori ini: git clone <url-repositori-anda>
    2. Buka file index.html langsung di browser web apa pun.
    3. Versi live dari website ini dapat diakses melalui GitHub Pages (Ganti dengan URL GitHub Pages Anda setelah rilis).
